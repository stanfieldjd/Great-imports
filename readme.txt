One-time Eventbrite URL importer for collecting review candidates.

== Description ==

Great Imports starts with a simple one-time Eventbrite URL import. It collects public schema.org Event JSON-LD from an Eventbrite event page and stores the result as an internal review candidate.

This version does not schedule recurring imports, does not directly publish Events Manager events, and does not geocode.
